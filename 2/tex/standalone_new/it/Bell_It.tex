\chapter{Théorème de Bell}

\section{Introduction}

Le théorème de Bell est l'un des résultats les plus profonds de la physique quantique. Il démontre que les corrélations quantiques ne peuvent pas être expliquées par des variables cachées locales.

\section{Les inégalités de Bell}

L'inégalité de Bell originale :

\begin{equation}
	|E(a,b) - E(a,c)| \leq 1 + E(b,c)
\end{equation}

où $E(a,b)$ est la corrélation entre les mesures avec les paramètres $a$ et $b$.

\section{Violation quantique}

La mécanique quantique prédit :

\begin{equation}
	E(a,b) = -\cos(\theta_{ab})
\end{equation}

Cette prédiction viole les inégalités de Bell pour certains angles.

\begin{insight}[title=Signification]
	La violation des inégalités de Bell prouve que :
	\begin{itemize}
		\item La réalité quantique est non-locale
		\item Les corrélations quantiques sont instantanées
		\item Les variables cachées locales sont impossibles
	\end{itemize}
\end{insight}

\section{Perspective T0}

Dans la théorie T0, la non-localité quantique est une propriété fondamentale du champ $\xi$ :

\begin{equation}
	\text{Non-localité} \leftrightarrow \text{Champ } \xi \text{ global}
\end{equation}

Le champ $\xi$ connecte instantanément tous les points de l'espace-temps.
