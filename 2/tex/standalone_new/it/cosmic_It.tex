\chapter{Phénomènes Cosmiques}

\section{Introduction}

L'univers présente de nombreux phénomènes à grande échelle. La théorie T0 offre une explication unifiée de ces phénomènes.

\section{Le fond diffus cosmologique (CMB)}

\begin{equation}
	T_{CMB} = 2,725 \, \text{K}
\end{equation}

\begin{insight}[title=Interprétation T0]
	Le CMB n'est pas un vestige du Big Bang, mais une manifestation du champ $\xi$ :
	\begin{equation}
		T_{CMB} = \frac{16}{9} \xi^2 \cdot E_\xi
	\end{equation}
\end{insight}

\section{Décalage vers le rouge}

Le décalage vers le rouge cosmologique :

\begin{equation}
	z = \frac{\lambda_{obs} - \lambda_{émis}}{\lambda_{émis}}
\end{equation}

Dans T0, cela résulte de l'interaction avec le champ $\xi$, pas de l'expansion.

\section{Matière sombre}

Les effets attribués à la matière sombre sont expliqués par le champ $\xi$ :

\begin{itemize}
	\item Courbes de rotation des galaxies
	\item Lentilles gravitationnelles
	\item Formation des grandes structures
\end{itemize}

\section{Énergie sombre}

Dans un univers statique T0, l'énergie sombre n'est pas nécessaire.
