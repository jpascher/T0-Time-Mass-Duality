\chapter{Unités SI}

\section{Introduction}

Le Système International d'Unités (SI) définit les unités de base de la physique. La théorie T0 offre une perspective unifiée sur ces unités.

\section{Unités de base}

\begin{enumerate}
	\item Mètre (m) - longueur
	\item Kilogramme (kg) - masse
	\item Seconde (s) - temps
	\item Ampère (A) - courant électrique
	\item Kelvin (K) - température
	\item Mole (mol) - quantité de matière
	\item Candela (cd) - intensité lumineuse
\end{enumerate}

\section{Unités naturelles T0}

Dans la théorie T0, nous utilisons les unités naturelles :

\begin{equation}
	\hbar = c = k_B = 1
\end{equation}

\begin{insight}[title=Conversion]
	Le paramètre $\xi$ permet de relier les unités naturelles aux unités SI :
	\begin{equation}
		[L]_{SI} = \xi^n \cdot [L]_{nat}
	\end{equation}
\end{insight}

\section{Constantes fondamentales}

Les constantes fondamentales en unités SI et leur expression en termes de $\xi$ :

\begin{enumerate}
	\item Vitesse de la lumière : $c = 299\,792\,458 \, \text{m/s}$
	\item Constante de Planck : $\hbar = 1,055 \times 10^{-34} \, \text{J}\cdot\text{s}$
	\item Charge élémentaire : $e = 1,602 \times 10^{-19} \, \text{C}$
\end{enumerate}
