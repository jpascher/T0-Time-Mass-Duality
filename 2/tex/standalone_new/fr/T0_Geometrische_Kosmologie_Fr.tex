\chapter{Cosmologie Géométrique}

\section{Introduction}

La cosmologie géométrique étudie la structure à grande échelle de l'univers. La théorie T0 offre une approche géométrique unifiée.

\section{Géométrie de l'univers}

Trois géométries possibles :

\begin{enumerate}
	\item Plate ($k = 0$)
	\item Sphérique ($k > 0$)
	\item Hyperbolique ($k < 0$)
\end{enumerate}

\begin{insight}[title=Géométrie T0]
	Dans T0, la géométrie est déterminée par le champ $\xi$ :
	\begin{equation}
		R = f(\xi)
	\end{equation}
	où $R$ est le rayon de courbure.
\end{insight}

\section{Courbure et matière}

L'équation de Friedmann modifiée :

\begin{equation}
	H^2 + \frac{k}{a^2} = \frac{8\pi G}{3}\rho + \Lambda_\xi
\end{equation}

où $\Lambda_\xi$ est la contribution du champ $\xi$.

\section{Structure fractale}

L'univers T0 a une structure fractale à toutes les échelles.
