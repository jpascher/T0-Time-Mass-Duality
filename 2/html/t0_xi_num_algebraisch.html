🔢 ξ_num — Algebraische Resonanzstruktur im Zahlenraum

Primzahlen sind basissystemunabhängig — wie natürliche Einheiten in der Physik.
Die Frage: Hat die Periodenstruktur von ℤ/Nℤ eine universelle Verteilung
die analog zur ξ-Hierarchie der FFGFT strukturiert ist?
Alle Arithmetik exakt via BigInt · keine Dezimalsystem-Artefakte.
Algebraische Grundlage (basisunabhängig):
Für N = p·q gilt: Carmichael-Funktion λ(N) = lcm(p−1, q−1).
Die kleinstmögliche Periode r = ord_a(N) teilt λ(N).
Definiere: ξ_num(N) = λ(N) / N — dimensionslos, repräsentationsunabhängig.
Für harte Primzahlen (RSA): ξ_num ≈ 1/2 (große Periode, schwer).
Für glatte Primzahlen: ξ_num ≪ 1 (kleine Periode, leicht).

Hypothese: Die Verteilung von ξ_num über zufällige semiprime N zeigt eine Resonanzstruktur — diskrete Häufungspunkte statt Gleichverteilung — analog zur ξ-Stufenhierarchie der FFGFT.
