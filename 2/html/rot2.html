Präzise CMB-Berechnungen in der T0-Theorie: Vollständiges Framework

1. Grundgleichungen des T0-CMB-Modells

A. Hintergrundevolution

\[\left(\frac{a'}{a}\right)^2 = \frac{8\pi G}{3}a^2\rho_{tot} \times \left[1 + \xi\left(\frac{T}{T_0} - \frac{1}{6}\left(\frac{T'}{\mathcal{H}T_0}\right)^2\right)\right]\]

wobei:

  • \(\mathcal{H} = a'/a\): Konformer Hubble-Parameter
  • \(\rho_{tot} = \rho_\gamma + \rho_b + \rho_T\): Gesamtdichte

B. T-Feld-Dynamik

\[T'' + 2\mathcal{H}T' + a^2\frac{\partial V}{\partial T} = \xi a^2 \rho_\gamma\]

C. Photon-Boltzmann-Gleichung

\[\frac{d\Theta}{d\eta} + ik\mu\Theta + \frac{d\Phi}{d\eta} + ik\mu\Psi = \tau'(\Theta_0 - \Theta + \mu v_b) + \xi \frac{T'}{T_0}(\Theta - \mu^2\Pi)\]

mit \(\Pi = \Theta_2 + \Theta_{P0} + \Theta_{P2}\): Polarisationsterme

2. T-Feld-Parameter und Kosmologie

T-Feld-Masse \(m_T\)

\[m_T = \sqrt{2\xi} H_0 \approx 1.6 \times 10^{-33} \text{ eV}\]

Strahlungsdichte \(\rho_{rad}\)

\[\rho_{rad}(z) = \rho_{rad,0}(1+z)^4 \left[1 + \xi \ln\left(\frac{1+z}{1100}\right)\right]\]

Hubble-Parameter \(H(z)\)

\[H(z) = H_0 \sqrt{\Omega_r(1+z)^4 + \Omega_m(1+z)^3 + \Omega_T} \times \left[1 + \xi f(z)\right]\] \[\text{mit } f(z) = 1 - \frac{1}{2}(1+z)^{-1/2}\]

4. Numerische Lösung (Kernimplementierung)

class T0CMBSolver:
    def __init__(self, xi=1.33e-4, Nk=200, lmax=3000):
        self.xi = xi
        self.k_grid = np.logspace(-5, 2, Nk)  # Mpc^-1
        self.lmax = lmax
        
    def solve_background(self):
        # Löse Friedmann + T-Feld-Gleichungen
        odeint(dydη, [a, T, T'], η_span)
        
    def solve_perturbations(self):
        for k in self.k_grid:
            # Boltzmann-Hierarchie bis lmax
            solve_coupled_ode(Θ_l, δ_b, v_b, Φ, Ψ)
            
    def compute_spectra(self):
        # Lineare Integration über k
        Cl = integrate(Δ_l(k)*Δ_l(k)*PΨ(k), k)
        return Cl_TT, Cl_EE, Cl_TE

7. Zusammenfassung

  • Testbare Vorhersagen für zukünftige Experimente
