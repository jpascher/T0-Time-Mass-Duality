  \item \textbf{The data fit -- via the cosmological degeneracy} (Doc.~267): FFGFT

\section{The static redshift}
% ============================================================

In the static universe a photon loses energy along the light path according to
\[
  \frac{dE}{dx} = -\xi\,E
  \qquad\Longrightarrow\qquad
  E(x) = E_0\,e^{-\xi x}.
\]
The linear dependence on $E$ is decisive: it makes the \emph{leading} effect
frequency-independent. The redshift is
\[
  z = \frac{E_0 - E(x)}{E(x)} = e^{\xi x} - 1,
  \qquad
  \boxed{\,1+z = e^{\xi x}\,},
  \qquad z \approx \xi\,x \ \text{(for small } z).
\]
Comparison with the Hubble law $z = H_0 d/c$ gives (natural units, $c=1$) the
identification
\[
  H_0^{\text{T0}} = E_H/\hbar,\qquad E_H = E_0\,\xi^{41/4} = 1{,}41\times10^{-33}\,\text{eV}
  \;\Rightarrow\; H_0^{\text{T0}} \approx 66{,}2\,\text{km/s/Mpc}
\]
(Doc.~064/026; the exponent $41/4$ is not derived from first principles,
Doc.~190~R20). The \emph{light-travel time} to an object at redshift $z$ is therefore
\[
  \boxed{\,\tau(z) = \frac{\ln(1+z)}{H_0^{\text{T0}}}\,}.
\]
There is \emph{no} finite cosmic age and \emph{no} Big Bang: $\tau$ grows without
bound as $\ln(1+z)$.

\section{Achromatic, not wavelength-dependent (correction)}
% ============================================================

Earlier documents (Doc.~004, 025, 030, 039, 041, 053, 061, 063, 068, 081) stated a
\emph{wavelength-dependent} redshift (e.g.\ $z(\lambda_0)\propto\lambda_0$,
\enquote{UV stronger than radio}). That presentation is \textbf{superseded}.

The mature FFGFT mechanism is \emph{fractal path-lengthening}: the path the signal
travels through the recursive time geometry is stretched by a factor that depends
only on distance/geometry -- \emph{not} on the photon wavelength. Hence
$1+z = e^{\xi x}$ holds for \emph{all} wavelengths with the same factor: the redshift
is \textbf{achromatic}.

\textbf{Evidence:} finite-element simulations of the fractal path-lengthening yield
\emph{no} wavelength-dependent redshift. This is also physically compelling: an
\emph{energy-/wavelength-specific} loss term would be chromatic, but a
\emph{geometric} stretch factor acts on wavelength \emph{and} pulse duration in the
same ratio (as do gravitational and standard-cosmology redshift, also achromatic).

\textbf{Consequence for observation:} the cosmological redshift is achromatic to high
precision -- FFGFT therefore gets this \emph{right}. The earlier
\enquote{wavelength-dependent discriminator} drops out; it moves from the
\enquote{test} column to the \enquote{fits} column. The correction is recorded in
Doc.~190 as \textbf{C6}; the affected documents are listed there.

\section{The $z\to$time mapping is model-dependent}
% ============================================================

Only the \emph{redshift} $z$ (and the quantity measured at that $z$) is
model-independent. The conversion $z\to$time, by contrast, is a modelling choice:

\begin{itemize}
  \item \textbf{$\Lambda$CDM} computes it via $a(t)$ (Friedmann integral). The
        look-back time levels off at $t_0 \approx 13{,}8$\,Gyr -- the
        \enquote{Big Bang ceiling}. That ceiling exists \emph{only because
        expansion is assumed}.
  \item \textbf{FFGFT} computes it via $\tau = \ln(1+z)/H_0^{\text{T0}}$. There is
        no ceiling; $\tau$ grows without bound.
\end{itemize}

\section{How do the data fit? -- the cosmological degeneracy}
% ============================================================

Doc.~267 establishes the central fact: FFGFT and $\Lambda$CDM are
\emph{observationally degenerate} on the standard tests. Whatever produces a given
$z(d)$ relation produces the same fluxes, angles and time dilations.

\begin{itemize}
  \item \textbf{Supernova time dilation:} distant SN~Ia run stretched,
        $b\approx1$ (DES: $b=1{,}003\pm0{,}011$). The fractal path-lengthening
        stretches wavelength \emph{and} pulse duration in the same ratio
        $\Rightarrow b=1$. The test only excludes \emph{non-dilating} tired-light
        ($b=0$) -- FFGFT predicts $b=1$ and is unaffected.
  \item \textbf{BAO angular scales:} the non-linear temporal recursion curves
        $z(d)$ like an accelerated expansion $\Rightarrow$ the same angles. Caveat
        R20: BAO uses $D_M/r_s$ with $r_s$ from $\Lambda$CDM -- model vs.\ model.
  \item \textbf{CMB peaks:} both reproduce the peak structure; $\xi$ plays no role
        for the peaks; both equally circular (Doc.~267/268).
  \item \textbf{Lilly--Madau / SFRD:} the $z$-space signal (rise to $z\sim2$, then
        fall) is model-independent and in FFGFT astrophysical galaxy evolution over
        light-travel time. The SFRD \emph{values} (M$_\odot$/yr/Mpc$^3$), however,
        are doubly $\Lambda$CDM-derived -- via the luminosity distance $d_L(z)$ and
        the comoving volume. A fair FFGFT test re-derives them with $d(z)$ and volume
        from $1+z=e^{\xi x}$.
\end{itemize}

\textbf{Honest status:} the degeneracy cuts both ways. The data \emph{fit} FFGFT (no
false prediction, achromatic as observed), but they do not \emph{compel} it. The case
for FFGFT therefore rests on \emph{parsimony} (no $\Lambda$, no dark energy, no $H_0$
tension -- Doc.~064) and on deriving the constants from $\xi$ -- not on a single
discriminator.

\section{No Big Bang -- the direction of development reverses}
% ============================================================

FFGFT has no Big Bang. The apparent high-redshift \enquote{onset} -- the fall of the
SFRD and the supposed beginning of the first galaxies at $z\sim14$ -- is in FFGFT
\emph{not} a temporal beginning but a \emph{geometric turning point} (Doc.~263).

The observed quantity is the cumulative fractal path-lengthening
$z_{\text{obs}} = (L_{\text{eff}}-d)/d$ (Doc.~026/263). For small $z$ it is linear,
$z(d) = \kappa\,d$ with $\kappa = E_H/(\hbar c)$ -- the slope $\Lambda$CDM names
\enquote{$H_0$}. Near the cosmological maximum scale $R_H$, however, the $z(d)$ curve
has an \emph{extremum}: FFGFT \emph{computes} there $z_*\approx 875$ (Doc.~267,
reference value; caveat R20) -- the same surface $\Lambda$CDM reads as
recombination/decoupling at $z\approx 1089$. The $\Lambda$CDM value $1089$ is thus
only the expansion reading; the FFGFT-own value is $875$, \emph{not} $1100$.
\emph{Beyond} $R_H$ one has $z\to\infty$ -- with no particle horizon and no Big Bang.

\textbf{What is $z_*$ from the FFGFT view?} It is \emph{not} a recombination time.
FFGFT has no hot early phase, no photon-baryon plasma, no decoupling. The CMB is the
\emph{stationary quantum ground state} of the $\Phi$ field (Doc.~025), and its acoustic
peaks are discrete T$^4$ \emph{lattice resonances} (analogous to the emission lines of
an atom or phonon dispersions in a crystal) -- \emph{not} frozen sound waves of a
primordial plasma. The numerical value itself is a \emph{geometric ratio} (Doc.~267):
\[
  z_* \approx \left(\frac{\lambda_e}{L_0}\right)^{1/\ln(1/\xi)} \approx 875,
\]
with the reduced Compton wavelength $\lambda_e$ and the sub-Planck length
$L_0=\xi\,\ell_P$ -- i.e.\ a \emph{resonance / structure scale} of the T$^4$ lattice,
not a thermal event in time. (Caveat R20: the exponent $1/\ln(1/\xi)$ is not yet derived
from the T$^4$ geometry; $z_*$ is currently a reference value of the right order of
magnitude, not a derived FFGFT result.)

Thus \textbf{the apparent direction of development reverses at the maximum scale}:
what $\Lambda$CDM reads as \enquote{earlier = faster development up to a beginning}
is in FFGFT the approach to the geometric extremum near $R_H$. The high-redshift fall
of the SFRD is therefore the approach to this turning point, \emph{not} a beginning.
$H_0$ is emergent, not a fundamental parameter -- in the compact $T^4$ form there is
no $H_0$ at all (Doc.~263).

Symmetrically open is only the forward derivation of the exponent $41/4$ from $\xi$
(Doc.~190~R20) -- missing for every framework, not only FFGFT.
