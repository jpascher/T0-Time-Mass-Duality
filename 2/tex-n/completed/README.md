# Book PDF Output Directories

This directory contains subdirectories for compiled book PDFs:

## Directory Structure

- **deutsch/** - German book PDFs (Teil1_De.pdf, Teil2_De.pdf, Teil3_De.pdf, Test_Landscape_De.pdf)
- **english/** - English book PDFs (Teil1_En.pdf, Teil2_En.pdf, Teil3_En.pdf, Test_Landscape_En.pdf)

## Note

The .tex source files remain in the parent directory (completed/) for compilation.
The subdirectories are for organizing the final PDF outputs by language.
