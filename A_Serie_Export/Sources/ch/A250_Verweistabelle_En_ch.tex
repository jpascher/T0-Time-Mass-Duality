\section{Purpose and Limits of This Document}

\textbf{[STIPULATION]} This document is a \emph{reference table}, not a register.
It assigns to each A-document the legacy documents it subsumes. It generates no
correction entries -- the A-series maintains no register of its own (A010).

The distinction matters. A register books corrections with date and status; that
is solely the role of Dok.~190, and remains the author's responsibility. This
table only refers -- it says where in the legacy corpus the material resides that
an A-document reorganises. References to legacy documents and existing register
entries are permitted; new entries are not.

outdated -- it means the A-series has not (yet) reorganised it.

\section{Open Questions}

\textbf{First, the coverage is incomplete.} The A-series orders part of the
corpus; the rest remains in the legacy. A complete transfer is not the goal of
this edition.

\textbf{Second, the mapping is not bijective.} Some legacy documents are subsumed
by several A-documents (e.g.\ 292, 101), some A-documents subsume several. The
table is a reference structure, not a renaming.
